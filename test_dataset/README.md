# MediaGuardX Test Dataset

200 face images for training and testing the deepfake detection model.

## Structure

```
test_dataset/
  fake/       100 AI-generated faces (thispersondoesnotexist.com)
  real/       100 real human faces (Unsplash, royalty-free)
```

## Model Performance

The EfficientNet-B0 model trained on this dataset achieves:
- **95% validation accuracy** (80/20 train/val split)
- **81.2% discrimination gap** (fake avg 15.5% vs real avg 96.8%)
- 100% recall on real images (zero false accusations)

## Expected Detection Results

| Folder | Expected Label | Trust Score Range |
|--------|---------------|-------------------|
| `fake/` | Deepfake / Suspicious | < 40 (Deepfake) or 40-69 (Suspicious) |
| `real/` | Authentic | >= 70 |

## Usage

Upload any image through the MediaGuardX dashboard to test detection.

To retrain the model:
```bash
cd backend
python ml/train_model.py --data_dir ../test_dataset --epochs 30 --batch_size 16
```

## Sources

- **Fake faces**: Generated by StyleGAN via thispersondoesnotexist.com (1024x1024)
- **Real faces**: Royalty-free photographs from Unsplash (640x640 crops)
